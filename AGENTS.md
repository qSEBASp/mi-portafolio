# AGENTS.md

## Project
Static portfolio site (HTML5/CSS3/vanilla JS). No build tooling, package manager, or test suite.

## Dev & Deploy
- **Local**: Open `index.html` directly, or use VS Code Live Server extension.
- **Deploy**: Push to `main` → GitHub Pages updates automatically (no build step required).

## Key Facts
- No lint, test, or typecheck commands exist.
- All static assets are in `assets/` (css/, js/, img/).
- `index.html` is the sole entrypoint.
